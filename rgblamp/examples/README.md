# Examples

- [simple.rs](./simple.rs) is the simplest example of how to use this library to set the color of a lamparray device.
- [filter_devices.rs](./filter_devices.rs) is an example of how to use `ReportDescriptorParser` to manually parse report descriptors. This is useful when you need to apply filters on what device to target and don't want to waste resources by parsing all devices beforehand.
