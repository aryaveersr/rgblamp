# Examples

- [basic.rs](./basic.rs)
  The simplest example of how to use this library to set the color of a lamparray device.

- [builder.rs](./builder.rs)
  Using `LampUpdateBuilder` to batch multiple updates together.

- [parse_descriptor.rs](./parse_descriptor.rs)
  Using `ReportDescriptorParser` to manually parse report descriptors. This is useful when you need to selectively parse devices.
